# EcoScore — Proof of Concept

Dieser PoC demonstriert die technische Kernlogik der EcoScore App für das Gurtenfestival 2026.

## Inhalt
- `ecoscore_poc.ipynb` — Jupyter Notebook mit Scoring-Logik, CO₂-Berechnung und Impact Report
- `ecoscore_impact_report.png` — Visualisierung des generierten Impact Reports

## Beschreibung
EcoScore ist eine Mobile App die nachhaltigkeitsrelevante Käufe am Gurtenfestival automatisch über das bestehende Cashless-Kartensystem erfasst und mit Reward-Punkten belohnt.

Der PoC zeigt:
1. Punkteberechnung pro Eco-Kategorie (Pfad A — Cashless-Karte)
2. Transport-Scoring (Pfad B — manuelle Anreise-Erfassung)
3. CO₂-Einsparung pro Kategorie
4. Persönlicher Impact Report
5. Visualisierung der Punkte und CO₂-Einsparungen

## Technologie
- Python 3
- Jupyter Notebook
- Matplotlib
